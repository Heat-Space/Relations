## Commands
* /love <username> (Kiss)
* /plunk <username>
* /marry <username>
/marry subcommands:
  * /marry -confirm (Confirms current marriage request)
  * /marry -reject (Rejects current marriage request)
  * /marry -info [username] (Checks if a user is married)
* /divorce
* /pat <username>
* /beat <username
* /date <username>
* /sex <username> (WTFFFFF)
